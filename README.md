# Guess The Flag
Данный репозиторий содержит код реализации веб-приложения/игры Guess The Flag
# Функционал
- Игрок последовательно получает карточки с картинками флагов,
есть поля для ввода страны флага, кнопка пропустить флаг(не знаю, что за флаг)
завершить сессию.
- В конце игры появляется отчёт со статистикой ответов, сколько флагов решил сегодня,
сколько решил правильно, сколько неправильно.
- Так же есть возможность получить детальный отчёт, где будут представлены флаги, на которые
пользователь ответил неверно.
- Если пользователь не знает флаг, то карточка переворачивается и показывается ответ
такой флаг будет занесён в user unknown flags.
## Режимы игры
### Random flags
- Просто список случайных флагов, игра заканчивается, либо когда игрок
принял решение закончить сессию, либо, когда прошёл все флаги
### Hard flags
- Список малоизвестных флагов
### User unknown flags
- Список тех флагов в перемешку, которые пользователь не смог распознать
## Leaderboard
- За каждый угаданный обычный флаг 1 очко
- За сложный 2
