# Guess The Flag - Frontend

React приложение для игры "Угадай флаг" с современным интерфейсом.

## 🚀 Технологии

- **React 18** с TypeScript
- **React Router** для навигации
- **Styled Components** для стилизации
- **Axios** для API запросов
- **LocalStorage** для хранения токенов и состояния игры

## 📁 Структура проекта

```
src/
├── components/
│   ├── Auth/
│   │   └── AuthForm.tsx          # Форма входа/регистрации
│   ├── Game/
│   │   ├── GameStart.tsx         # Экран начала игры
│   │   ├── GamePlay.tsx          # Основной игровой экран
│   │   └── GameResults.tsx       # Экран результатов
│   └── ProtectedRoute.tsx        # Компонент защищенных маршрутов
├── services/
│   └── api.ts                    # API сервис для работы с backend
├── types/
│   └── api.ts                    # TypeScript типы для API
└── App.tsx                       # Главный компонент приложения
```

## 🎮 Экраны приложения

### 1. Экран аутентификации (`/login`)
- Вход в систему
- Регистрация нового пользователя
- Автоматическое перенаправление после успешной аутентификации

### 2. Экран начала игры (`/game`)
- Приветственный экран
- Описание игры
- Кнопка "Начать игру"
- Кнопка выхода из аккаунта

### 3. Игровой экран (`/game/play`)
- Отображение флага страны
- Поле ввода для ответа
- Кнопка отправки ответа
- Показ результата (правильно/неправильно)
- Кнопки "Следующий вопрос" и "Завершить игру"
- Счетчик вопросов

### 4. Экран результатов (`/game/results`)
- Общий счет и процент правильных ответов
- Детальный список всех вопросов с результатами
- Кнопки "Новая игра" и "Главное меню"

## 🔧 Установка и запуск

### Предварительные требования
- Node.js (версия 14 или выше)
- npm или yarn

### Установка зависимостей
```bash
cd frontend
npm install
```

### Запуск в режиме разработки
```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена
```bash
npm run build
```

### Запуск тестов
```bash
npm test
```

## 🔗 API Integration

Приложение работает с backend API, который должен быть запущен на `http://localhost:8080`.

### Основные эндпоинты:
- `POST /auth/register` - регистрация
- `POST /auth/login` - вход
- `POST /game/start` - начало игры
- `POST /game/question` - получение вопроса
- `POST /game/answer` - отправка ответа
- `POST /game/end` - завершение игры

## 🔧 Конфигурация

### Environment Variables
Создайте файл `.env` в корне проекта для настройки:

```env
REACT_APP_API_URL=http://localhost:8080
```

### Настройка API URL
По умолчанию используется `http://localhost:8080`. Для изменения отредактируйте файл `src/services/api.ts`.
